## Trick
> This is because within a BST, all the elements in the left subtree are smaller than those within the right subtree
## Approach 
```python
class Solution:
	def searchBST(self, root: Optional[TreeNode], val: int) -> Optional[TreeNode]:
		  
		def dfs(root, val):
			  
			if root.val == val:
				return root
			
			if root.right and val > root.val:
				return dfs(root.right, val)
			
			if root.left and val < root.val:
				return dfs(root.left, val)
				
			if not root.left or not root.right:
				return
		
		return dfs(root, val)
```
